¿Qué es?

Clonezilla es un software libre de recuperación de una imagen creada de un sistema operativo, tiene diversas funcionalidades como crear una imagen del sistema o eliminarla o por ejemplo eliminar una partición.

Procedimiento de la práctica

Lo primero de todo es descargar la .iso de Clonezilla desde la pagina oficial de Clonezilla. Después con nuestro pendrive nos vamos a la herramienta de creación de disco de arranque y seleccionamos la ISO de Clonezilla y nuestro pendrive

Después de un tiempo estaría ya listo para iniciar. Apagamos el dispositivo y cuando se esté iniciando, presionamos la tecla f11 o suprimir (cambia dependiendo de S.O) E iniciamos el dispositivo mediante el pendrive

Hacemos unos ajustes de organización de las letras del teclado (mejor darle a automático), idioma y otras cosas más. Ahora viene lo importante,después de los ajustes iniciales nos aparecerá esta ventana

Como vamos a crear un disco / partición desde una imagen (esa imagen son los archivos actuales del dispositivo) Por lo cual seleccionamos la primera opción El siguiente paso seria elegir una partición, nuestro pendrive debe de estar dividido en 2. Una parte es la de Clonezilla y otra es el espacio libre para guardar la imagen

Seleccionaremos el modo Berlingar (o si sois unos pros Expert) en mi caso, voy a usar el modo beginner
Después, nos pide si queremos guardar el disco o las particiones de este.

Nos pedirá una comprobación del disco para ver si hay errores pero duraría mucho mas de lo normal Al terminar estos pasos empieza la creación de la imagen (este proceso puede tardar bastante)

Después de un buen rato ya estaría terminada la creación de la imagen del dispositivo P.D:Si sale algún error durante el ultimo proceso habrá que repetir todo de nuevo y revisar bien lo que se hace para dar con el error Lo que hemos hecho ha sido el proceso para crear una imagen del disco del dispositivo,ahora vamos a restaurar una imagen en dicho ordenador (es decir restaurar una imagen en el dispositivo)

Hacemos los mismos pasos que antes y llegará u momento en el que nos salte esta ventana
Ha detectado la imagen que hemos hecho antes, permitiendo restaurarla Seguimos avanzando y llegamos a esta parte
Como estamos restaurando dicha imagen debemos seleccionar la opción restoredisk para restaurar la imagen a disco local
Seleccionamos el disco de destino
Y luego nos pedirá el que disco vamos a recuperar dicha imagen

Elegimos si queremos rehacer la tabla de particiones o empezar de cero

Y ahora a esperar y esperar que no ocurra ningún error

Y ya estaría hecho
