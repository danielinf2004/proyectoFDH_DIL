En Windows

En mi caso yo voy a descargar EaseUS Data Recovery para Windows que es una herramienta para recuperar esos archivos que sin querer , lo has eliminado permanente.Esta tiene unos requisitos de 32MB de espacio en el disco duro y 1 GB de RAM para un correcto funcionamiento. Nos metemos en esta pagina para descargar EaseUS Data Recovery y le damos al botón de descargar gratis

Una vez descargado el archivo .exe lo ejecutamos y seguimos el proceso de instalación.(Este proceso puede tardar un poco).

Cuando la instalación haya terminado , ejecute el programa para poder recuperar los archivos eliminados deseados

Y ya estaría

En Ubuntu

Para instalarlo hay que descargar el archivo desde esta pagina y darle a "Ubuntu" y se empieza a descargar el archivo
